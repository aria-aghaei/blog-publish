About

One day I, an extremely wise and intelligent person, had a glorious idea and created this amazing website in like 30 minutes. It only took me half an hour to make this website because I have the power of theft. Did you know that taking open-source code is easy and you can basically do it whenever you want?

To pull this off you will need:

  • Basic command line skills
  • Bare minimum git skill
  • HTML and CSS skill (maybe, or you can just wing it)
  • That's pretty much it

The steps to making a good website are simple. First, git clone this repo and then just take the example page and css file and edit it to your liking. Then, cd into the website directory and do this:

$ npm install http-server -g
$ http-server

You can now browse to your website. Just type localhost:8080 into your browser's address box. However, the website can only be accessed by you, on your local machine!
